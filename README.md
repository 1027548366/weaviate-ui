# Weaviate-UI

![screenshot](screenshot.png)

Weaviate-UI is a web client for interacting with the Weaviate.

## Features

- Schema query
- Data search

## Usage

```bash
$ docker run -e WEAVIATE_URL=http://localhost:8091 -e WEAVIATE_API_KEYS=secret naaive/weaviate-ui:latest
```
